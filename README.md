Новый проект авто запуска

1. Создать чистый проект, не привязанный к языку
2. Создать build.gradle, добавить параметры из файла и запустить компиляцию
3. Создать .gitignore, добавить параметры из файла
4. ПКМ - new girectory - scr\test\java + resources
5. В java создаем класс, в котором будем запускать тесты
6. В resources создать logback-test.xml, добавить параметры из файла
